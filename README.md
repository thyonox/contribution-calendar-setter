# CCS
This is a tool to modify the contribution calendar.



## Prerequisites
1. use a private repository.
1. click contribution settings, select private contributions.

## Install
```shell
    npm i jsonfile
    npm i moment
    npm i random
    npm i simple-git
```
## Step
1. Initialize the node project.
    ```shell
        npm init -y
    ```
2. install dependencies
3. run index.js
